virtualenv running python 2.7 created by executing:
virtualenv --python=/usr/bin/python2.7 python27


activate python2.7 by typing:
source python27/bin/activate

to deactivate:
deactivate


to create new env specifying python version:
python3.6 -m venv python36
<python version> <idk> <idk> <filename>


to activate python 3.6 by typing:
python36/bin/activate


to install latest version of package:
pip install -U coremltools
***** "-U" flag installs latest version of package*****
